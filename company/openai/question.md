Coding: GPU Credits II
You are given an array of positive numbers credits, where each value represents GPU credits for a user/task. In one operation, you may pick an element x and replace it with x/2 (keep as a floating point number; no rounding required).

Return the minimum number of operations needed to make the sum of the array reduced to at most half of its original value.

Input (stdin)
Line 1: integer n, the length of the array
Line 2: n positive numbers credits[i]
Output (stdout)
Print a single integer: the minimum number of operations
Constraints
1 <= n <= 2 * 10^5
1 <= credits[i] <= 10^9
Example
Input:

3
5 19 8
Output:

3
Explanation:

Sum is 32, target is <= 16.
Always halve the current largest value: 19->9.5 (reduce 9.5), 9.5->4.75 (reduce 4.75), 8->4 (reduce 4). Total reduced 18.25, done in 3 ops.
